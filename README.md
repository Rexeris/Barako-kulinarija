# Barako kulinarija 🍲

**Barako kulinarija** – studentams (ir visiems taupantiems) skirtas kulinarinis puslapis su receptais, patarimais ir idėjomis, kaip skaniai pavalgyti neišleidžiant daug.  
Projektas kuriamas su **Laravel 12**.

## Funkcionalumas

- Receptų sąrašas ir recepto peržiūra
- Paieška ir filtrai (kaina / laikas / sudėtingumas)
- Kategorijos
- (Admin) receptų kūrimas / redagavimas / trynimas (CRUD)
- Patarimų skiltis („kaip pigiau / greičiau / skaniau“)
- Video pavyzdžiai

> Papildomai (jei spėsim): pirkinių sąrašas, „random receptas“, „pigių radinių“ skiltis.

## Tech stack

- **Laravel 12**
- PHP 8.x
- MySQL / SQLite (pagal komandos pasirinkimą)
- Node.js + NPM (fronto build’ui)
